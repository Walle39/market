# GMVM v6.1 黄金市场估值模型

## 模型架构

```
┌──────────────────────────────────────────────────────────────┐
│ 第一层：宏观驱动信号层（离散编码，高频代理）                 │
│ 因子：央行购金代理、M2高频预估、债务/GDP、Fed量化预期差、  │
│       TIPS、DXY                                             │
│ 输出：S_macro (-1.0 ~ +1.0)                                 │
└──────────────────────────┬───────────────────────────────────┘
                           ↓ (乘法)
┌──────────────────────────────────────────────────────────────┐
│ 第二层：市场验证系数（保留）                                  │
│ 因子：ETF持仓、金银比、AISC成本                               │
│ 输出：K_verif (0.5 ~ 1.5)                                    │
└──────────────────────────┬───────────────────────────────────┘
                           ↓ (乘法)
┌──────────────────────────────────────────────────────────────┐
│ 第三层：趋势动量系数（保留）                                  │
│ 因子：RSI、MACD背离、均线斜率、布林带                         │
│ 输出：K_trend (0.6 ~ 1.4)                                    │
└──────────────────────────┬───────────────────────────────────┘
                           ↓ (乘法)
┌──────────────────────────────────────────────────────────────┐
│ 第四层：动态流动性系数（保留矩阵）                            │
│ 因子：VIX、信用利差                                          │
│ 输出：K_liquidity (0.3 ~ 1.2)                                │
└──────────────────────────┬───────────────────────────────────┘
                           ↓ (乘法)
┌──────────────────────────────────────────────────────────────┐
│ 第五层：地缘条件化乘数（方案A）                               │
│ 因子：GPR指数、布伦特油价、CPI同比、Fed预期差                 │
│ 输出：K_geo (0.95 ~ 1.20)                                    │
└──────────────────────────┬───────────────────────────────────┘
                           ↓
┌──────────────────────────────────────────────────────────────┐
│ 最终综合信号 = S_macro × K_verif × K_trend × K_liquidity × K_geo │
│ 叠加强制规则覆盖                                              │
└──────────────────────────────────────────────────────────────┘
```

## 最终信号与仓位指导

| 最终信号范围 | 信号等级 | 操作建议 | 建议仓位 |
|--------------|:--------:|----------|:--------:|
| ≥ +0.60 | 🔥 强力看多 | 积极加仓，可适度杠杆 | 80%-100% |
| +0.35 ~ +0.60 | ✅ 看多 | 分批建仓，回调加仓 | 60%-80% |
| +0.15 ~ +0.35 | 🟡 偏多 | 持有为主，轻仓可增 | 40%-60% |
| -0.15 ~ +0.15 | ⚪ 中性 | 观望，不做方向性操作 | 20%-40% |
| -0.35 ~ -0.15 | 🟠 偏空 | 减仓，了结部分利润 | 10%-20% |
| ≤ -0.35 | 🔴 看空 | 清仓或对冲做空 | 0%-10% |

## 文件结构

| 文件 | 说明 |
|------|------|
| **gmvm_v61.py** | GMVM v6.1 主程序 |
| **gmvm_layer1_macro.py** | Layer 1: 宏观驱动信号层 (S_macro) |
| **gmvm_layer2_verif.py** | Layer 2: 市场验证系数 (K_verif) |
| **gmvm_layer3_trend.py** | Layer 3: 趋势动量系数 (K_trend) |
| **gmvm_layer4_liquidity.py** | Layer 4: 动态流动性系数 (K_liquidity) |
| **gmvm_layer5_geo.py** | Layer 5: 地缘条件化乘数 (K_geo) |

## 使用方法

```bash
# 运行GMVM v6.1模型
python3 gmvm_v61.py

# 可选：设置手动输入参数
export MANUAL_FED_EXPECTATION=-1  # Fed预期: -1鹰派, 0中性, 1鸽派
python3 gmvm_v61.py
```

## 强制规则

| 规则ID | 触发条件 | 强制动作 |
|--------|----------|----------|
| OR-03 | VIX≥35且黄金美元同跌 | 最终信号≤-0.50 |
| OR-01 | 极度超买+顶背离 | 最终信号≤-0.20 |
| OR-02 | 极度超卖+底背离 | 最终信号≥+0.30 |
| 结构牛底线 | 央行连续两季净购金>300吨 | 下限锚定+0.15 |
| OR-05 | 验证层>0.8且S_macro>0 | 信号≥+0.15 |
| OR-04 | 央行购金>300吨且金价低于高点10% | 额外+0.10 |
